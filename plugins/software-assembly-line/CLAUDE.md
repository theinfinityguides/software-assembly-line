# Software Assembly Line Plugin

AI agents for automated software development on Effect-TS/Bun codebases.

## Commands

- `/software-assembly-line:plan` - Transform PRD into Linear issues
- `/software-assembly-line:work` - Pick and implement next story
- `/software-assembly-line:review` - Multi-agent code review
- `/software-assembly-line:compound` - Capture session learnings
- `/software-assembly-line:scaffold` - Generate CRUD scaffolds
- `/software-assembly-line:workflow` - Generate Effect Cluster workflows

## Agents

This plugin provides specialized agents for:
- Planning (PRD analysis, story generation, dependency linking)
- Plan Review (Effect feasibility, type complexity, monorepo impact)
- Code Review (Effect patterns, type strictness, security, performance)
- Compounding (pattern documentation, error solutions)
